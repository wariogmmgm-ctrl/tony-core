# App TONI

App Android para capturar fotos, escribir notas y sincronizar entre dispositivos.

## Características
- 📷 Tomar fotos de Toni
- ✍️ Escribir notas
- 📤 Sincronizar automáticamente en GitHub
- 📱 Funciona en dos Oppos sincronizados

## Compilar a APK

1. Descarga Android Studio
2. Clona este repositorio
3. Abre el proyecto en Android Studio
4. Conecta tu Oppo o abre un emulador
5. Click en "Run" (Shift + F10)

## Estructura

```
tony-core/
├── build.gradle (configuración del proyecto)
├── AndroidManifest.xml (permisos y actividades)
├── MainActivity.java (lógica principal)
├── activity_main.xml (interfaz)
└── README.md (este archivo)
```

- Cámara
- Almacenamiento
- Internet (para sincronizar)
